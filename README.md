# Pizza Sales Analysis Dashboard 

Project Overview

This project focuses on analyzing pizza sales data to uncover business insights, improve decision-making, and understand customer behavior.
The dashboard is built using **Power BI** with support from **SQL and Python** for data processing and analysis.

Objective

* Analyze overall sales performance
* Identify best and worst-selling pizzas
* Understand customer ordering patterns
* Track revenue trends over time

Tools & Technologies Used

* Power BI → Data visualization & dashboard creation
* SQL → Data querying and KPI calculations
* Python (Pandas, Plotly,Numpy) → Data cleaning & exploratory analysis

Dataset Information

The dataset contains:

* Order ID
* Pizza Name
* Category 
* Size 
* Quantity
* Price
* Order Date

Key Performance Indicators (KPIs)

* Total Revenue → Total sales generated
* Total Orders → Number of unique orders
* Total Quantity Sold → Total pizzas sold
* Average Order Value → Revenue per order
* Average Pizzas per Order

Dashboard Features

1. Sales Trends

* Daily order trends
* Monthly sales trends
* Peak sales days identification

2. Category Analysis

* Sales by pizza category
* Contribution of each category to total revenue

3. Size Analysis

* Revenue distribution by pizza size
* Most preferred pizza size

4. Best & Worst Sellers

* Top 5 pizzas by revenue
* Top 5 pizzas by quantity
* Bottom 5 pizzas by revenue

5. Funnel / Order Distribution

* Number of pizzas ordered per category
* Customer ordering behavior
